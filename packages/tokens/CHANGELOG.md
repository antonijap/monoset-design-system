# @monoset/tokens

## 0.2.0

### Minor Changes

- Shadows are now layered: a tight contact shadow paired with a soft ambient one at each level, so elevated surfaces read as gently lifted.

  New tokens: --ring for focus outlines (follows --fg1, so it stays readable in dark mode) and --z-dropdown. A pre code reset stops code blocks from inheriting inline-code chrome.

## 0.1.0

### Minor Changes

- Initial public release. Monoset is a minimal, monotone, unopinionated design system: tokens, Framer Motion presets, and a React component kit wrapping Radix primitives. See [monoset.design](https://monoset.design) for docs.
