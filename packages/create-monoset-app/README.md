# create-monoset-app

Scaffold a new app pre-configured with [Monoset](https://monoset.design) tokens and components.

## Usage

```bash
npm create monoset-app@latest my-app
cd my-app
npm install
npm run dev
```

Or scaffold into the current directory:

```bash
mkdir my-app && cd my-app
npm create monoset-app@latest .
```

## What you get

A minimal Vite + React 19 project with:

- `@monoset/tokens` — the CSS variables
- `@monoset/react` — the component kit
- `framer-motion` — motion presets
- A working example page with Card, Field, Input, Switch, and Buttons
